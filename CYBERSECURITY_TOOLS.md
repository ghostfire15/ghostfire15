# Awesome Cybersecurity Tools Collection

Collection des meilleurs outils open-source de cybersécurité sur GitHub.

## TOP 10 Outils

1. [the-book-of-secret-knowledge](https://github.com/trimstray/the-book-of-secret-knowledge) - 229K ⭐ - Collection de listes et manuels
2. [sherlock](https://github.com/sherlock-project/sherlock) - 85K ⭐ - Recherche de comptes sociaux
3. [hackingtool](https://github.com/Z4nzu/hackingtool) - 77K ⭐ - Outil tout-en-un
4. [worldmonitor](https://github.com/koala73/worldmonitor) - 57K ⭐ - Tableau de bord OSINT
5. [x64dbg](https://github.com/x64dbg/x64dbg) - 48K ⭐ - Debugger Windows
6. [funNLP](https://github.com/fighting41love/funNLP) - 81K ⭐ - Ressources NLP
7. [sniffnet](https://github.com/GyulyVGC/sniffnet) - 39K ⭐ - Analyseur réseau
8. [web-check](https://github.com/lissy93/web-check) - 33K ⭐ - Analyse de sites web
9. [maigret](https://github.com/soxoj/maigret) - 33K ⭐ - OSINT username search
10. [nuclei](https://github.com/projectdiscovery/nuclei) - 29K ⭐ - Scanner de vulnérabilités
